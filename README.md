# Projet pokéapi

## Objectif

L'objectif de ce projet est de créer une liste des différents pokémons et de pouvoir accéder à leurs informations en utilisant Vue.js.

## Usage
```
cd pokeapi
npm run build
```

Ouvrir le fichier dist/index.html
